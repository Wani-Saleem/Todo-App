Todo App
The Todo App is a simple web application that allows users to create, manage, and organize their tasks.

Features
Add new tasks
Mark tasks as completed
Delete tasks
Filter tasks by status (completed or active)
Clear all completed tasks
Technologies Used
HTML
CSS
JavaScript
Usage
